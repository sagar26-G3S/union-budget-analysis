# 📊 Union Budget Analysis Dashboard

An interactive data analytics web application that visualizes and explores Indian Union Budget data using Tableau dashboards embedded inside a professional Flask-based web interface.

This project combines:

- Tableau Public (Data Visualization & Analytics)
- Flask (Web Application Framework)
- HTML/CSS (Professional UI Design)
- Embedded Interactive Storytelling

---

## 🚀 Project Overview

The **Union Budget Analysis Dashboard** provides insights into fiscal allocations, department-wise budget distribution, category-wise analysis, and scheme-level comparisons.

Users can interactively explore:

- Total Allocation Insights
- Category-wise Budget Distribution
- Department-wise Analysis
- Top Schemes Visualization
- Trend Analysis through Tableau Stories

The application integrates Tableau dashboards inside a modern enterprise-style web UI for seamless analytics experience.

## 📊 Tableau Integration

The dashboards and story visualizations are created using Tableau Public.

Tableau embed code is integrated inside `index.html` using iframe/object embedding.

Users can:

- Navigate through story scenes
- Apply filters
- Interact with visual analytics
